# @todo/lang-elixir

Elixir + Phoenix + Absinthe api. And maybe web app?

In boilerplate phase

## Stack

- lang: `elixir`
- framework: `phoenix`
- graphql: `absinthe`
- orm: `ecto`
- db: `postgres`

### Requirements

- `elixir`
- `hex`
- `phx_new`
- `inotify-tools` (for some reason...)
